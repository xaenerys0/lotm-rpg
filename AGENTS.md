# Writing GitHub Issues

When filing or writing up a GitHub issue (bug report or feature/task request), follow `.claude/skills/write-github-issue/SKILL.md`. It defines the required content, the matching issue templates in `.github/ISSUE_TEMPLATE/`, and the scripts in `.claude/skills/write-github-issue/scripts/` that handle all GitHub interaction (duplicate search + issue creation via `gh`).
